Place your child theme language files in this directory. 
The file names should follow this format: it_IT.po & it_IT.mo (example for Italian translation). Your file names should not include the theme slug.

Please visit the following links to learn more about translating WordPress themes:

http://codex.wordpress.org/Translating_WordPress
http://codex.wordpress.org/Function_Reference/load_theme_textdomain
